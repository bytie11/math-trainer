# 🧠 Tradermath Math Trainer

A professional **mental math trainer** designed for traders and professionals who want to sharpen their arithmetic speed and accuracy.

![License](https://img.shields.io/badge/license-MIT-blue.svg)

## Features

- **4 operations** — Addition, Subtraction, Multiplication, Division
- **3 number types** — Integers, Decimals, Fractions
- **Multiple choice & open answer** modes
- **Timed sessions** (1–10 minutes) with configurable question counts
- **Performance analytics** — score trends, speed trends, accuracy breakdown (Chart.js)
- **Adaptive difficulty** — questions get harder as you improve
- **Streak counter**, difficulty badges, keyboard shortcuts
- **Dark mode** — respects system preference, toggleable
- **Sound effects** — optional Web Audio API tones
- **Confetti** on personal bests 🎉
- **Fully accessible** — semantic HTML, keyboard navigation, ARIA live regions
- **Responsive** — works on mobile (375px+)

## Quick Start

**No build tools or server required.** Just open the HTML file in your browser:

```bash
open math-trainer.html
```

Or double-click `math-trainer.html` in your file manager.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `1`–`5` | Select answer tile (multiple choice mode) |
| `Enter` | Submit answer (open mode) |
| `Escape` | End session (with confirmation) |
